# SkyCast

A modern weather application built with React that provides real-time weather information for cities around the world.

## Features

- Search weather by city name
- Real-time temperature updates
- Humidity and wind information
- Feels-like temperature
- Responsive design
- Clean and user-friendly interface

## Tech Stack

- React.js
- JavaScript
- Material UI
- OpenWeather API
- CSS

## Installation

```bash
git clone <repository-url>
cd skycast
npm install
npm run dev
```

## Environment Variables

Create a `.env` file and add:

```env
VITE_API_KEY=your_api_key
VITE_API_URL=your_api_url
VITE_HOT_WEATHER=weather_image_like_summer
VITE_COLD_WEATHER=weather_image_like_winter
VITE_RAINY_WEATHER=weather_image_like_rainy
```
